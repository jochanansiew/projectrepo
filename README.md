# Project Repository

## Overview
Repository created for assignment.
Serves as an introduction to repository setup, file management, and version tracking using Git.

## Objectives

The key objectives of this project are to:
- Initialise and configure a local Git repository
- Create and manage project files using version control
- Practice staging and committing changes with meaningful commit messages
- Understand the basic workflow of Git-based project management
